### Feature: Strava dashboard

User can:

- authenticate with Strava
- see recent activities
- open activity detail
- export training summary as TOON JSON

Metrics displayed:
- distance
- pace
- avg HR
- elevation
- training load
- gear
- device name

### Feature: Strava webhook
User can:

- authenticate with Strava

The system can:
- fetch all latest 30 activity
- setup webhook call
- update the db and stats when triggered by webhook
